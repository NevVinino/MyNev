Descripción:
Diseño, desarrollo y despliegue de una plataforma web dinámica estilo catálogo multimedia para un creador de contenido de YouTube. Implementé una arquitectura basada en Supabase que actúa como un gestor de contenido dinámico y sistema de caché inteligente; esto permite actualizar todo el contenido visual de la página (videos de YouTube, biblioteca multimedia, PDFs) mediante la base de datos sin necesidad de modificar el código fuente. Además, automatiza la sincronización de videos recientes y directos optimizando las cuotas de la API de YouTube.
